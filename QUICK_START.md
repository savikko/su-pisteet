## What It Does

- Connects to your SQL Server
- Gets the latest skating results
- Uploads them to the website
- Shows you when it's done

## If Something Goes Wrong

**"Cannot connect to SQL Server"**
- Make sure SQL Server is running

**"Access denied"**
- Right-click the shortcut
- Select "Run as administrator"

**Other problems?**
- Contact your IT person
- Show them the error message in the window

## How Often Should I Run This?

Run the shortcut whenever:
- New results are added to the database
- You want to update the website with latest data
- After a competition day

It's safe to run multiple times - it will replace old data with new data.
